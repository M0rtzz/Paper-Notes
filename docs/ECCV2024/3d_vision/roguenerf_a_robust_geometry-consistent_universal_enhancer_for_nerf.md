# RoGUENeRF: A Robust Geometry-Consistent Universal Enhancer for NeRF

**会议**: ECCV 2024  
**arXiv**: [2403.11909](https://arxiv.org/abs/2403.11909)  
**代码**: [有](https://sib1.github.io/projects/roguenerf/)  
**领域**: 3D视觉  
**关键词**: NeRF增强, 几何一致性, 3D对齐, 非刚性精炼, 新视角合成

## 一句话总结

提出 RoGUENeRF，通过结合3D对齐、非刚性精炼和几何感知注意力机制，作为NeRF的后处理增强器，显著提升多种NeRF基线的渲染质量，同时保持视角一致性和对不准确相机标定的鲁棒性。

## 研究背景与动机

NeRF在高频纹理重建上存在局限：低频偏差、形状-辐射模糊性、不准确的相机标定导致细节模糊。现有增强方法要么是2D方法（如NeRFLiX），不感知场景几何，无法保持视角一致性；要么是3D方法（如Boosting View Synthesis），受限于不准确的几何和相机位姿。本文旨在结合两者优势：预训练通用增强函数 + 利用3D几何对齐的邻近训练图像信息。

## 方法详解

### 整体框架

1. **最近邻选择**：基于相机位置和朝向找到n个最近训练图像
2. **3D对齐**：利用深度图和相机位姿将训练图像特征重投影到新视角
3. **非刚性精炼**：轻量光流网络补偿几何误差
4. **几何感知注意力**：基于像素差异和相机距离调节对齐特征贡献
5. **特征融合+Uformer增强**：输出高质量图像

### 关键设计

**3D对齐**：使用深度图和相机内外参将邻近图像特征重投影到目标视角，通过可见性测试排除被遮挡区域。

**非刚性精炼**：在特征空间使用轻量迭代光流网络补偿相机位姿估计和NeRF几何的不准确，即使输入域差距大也能工作。

**几何感知注意力**：两级注意力——像素级（基于特征差异和深度差异）和相机级（基于相机位姿距离），共同调节邻近图像特征的贡献权重。

### 损失函数

$$L = |H_i - \hat{H}_i|_1 + 10^{-3}|\omega(\hat{H}_i) - \omega(H_i)|_1$$

L₁损失 + VGG-19感知损失。先在LLFF+NeRF上预训练，再在每个新场景上微调（<60分钟）。

## 实验关键数据

### 主实验

**360v2数据集各NeRF基线增强效果**：

| 模型 | PSNR↑ | SSIM↑ | LPIPS↓ |
|------|-------|-------|--------|
| MipNeRF360 | 28.26 | 0.805 | 0.230 |
| + NeRFLiX | 28.44 (+0.18) | 0.804 | 0.244 |
| **+ RoGUENeRF** | **28.89 (+0.63)** | **0.830** | **0.199** |
| Nerfacto | 26.11 | 0.716 | 0.327 |
| + NeRFLiX | 26.92 (+0.81) | 0.741 | 0.304 |
| **+ RoGUENeRF** | **27.45 (+1.34)** | **0.770** | **0.267** |
| NeRF (LLFF) | 26.57 | 0.817 | 0.239 |
| + NeRFLiX++ | 27.25 (+0.68) | 0.858 | - |
| **+ RoGUENeRF** | **27.67 (+1.10)** | **0.871** | **0.150** |

### 消融实验

**LLFF数据集NeRF基线消融（平均8场景）**：

|------|------|------|-------|
| NeRF基线 | 26.57 | 0.817 | 0.239 |
| + Uformer | 26.74 | 0.831 | 0.225 |
| + 3D对齐 | 26.84 | 0.840 | 0.208 |
| + 非刚性精炼 | 26.93 | 0.847 | 0.180 |

### 关键发现

- 在6个不同NeRF基线上一致提升所有指标，有效性广泛
- 在含噪声相机位姿时提升更大（Nerfacto中等噪声时+17.3 MUSIQ分）
- 非刚性精炼贡献最大的LPIPS改善（13.4%）
- 预训练策略能在1分钟微调后就获得显著提升

## 亮点与洞察

1. **通用性强**：适用于6种不同NeRF基线，无需修改底层模型
2. **鲁棒性好**：对不准确的相机标定具有鲁棒性，噪声越大提升越明显
3. **高效微调**：预训练后每场景仅需<60分钟微调
4. 3D+2D混合对齐策略巧妙解决了几何误差问题

## 局限性

- 需要存储训练图像和深度图用于推理
- 依赖NeRF的深度估计质量
- 对极度大角度变化的邻近图像效果有限

## 相关工作与启发

- NeRFLiX完全2D方法，不含几何信息，存在闪烁问题
- Boosting View Synthesis依赖像素完美对齐，实际中难以满足
- 启发：作为后处理的增强方案具有极高的实用价值和灵活性
